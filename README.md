Cicada Thermal Hub is a lightweight client–server application that centrally monitors CPU and GPU temperatures across multiple production machines in real time. 
A low-overhead client agent collects thermal data and sends it to a central server, providing efficient, scalable, and reliable monitoring with minimal resource usage.

This application relays in LibreHardwareMonitor to collect the thermal data from the CPU and GPU so is a requirement to have this opened in your PC to make it work DON´T CLOSE IT

You can also add sections to divide and organize the Monitoring Boxes.

The Temperature number of the components will change their color in case the temperature get higher than the stablished limit in the Settings. By default this temperature is set in 80°
